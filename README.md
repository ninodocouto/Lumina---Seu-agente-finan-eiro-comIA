# Lumina-Seu-agente-financeiro-com-IA
 # Resumo do Projeto

Este é um aplicativo de **finanças pessoais** que funciona por meio de conversas simples e naturais com o usuário.  
A proposta é eliminar a complexidade de planilhas e formulários, oferecendo uma experiência acessível e intuitiva, onde o usuário sente que tem controle total.

O app permite:

- **Registrar gastos via chat** em linguagem natural.
- **Classificar automaticamente** as transações.
- **Definir metas financeiras** e acompanhar progresso.
- Receber dicas de economia de um **Agente Financeiro**.
- Visualizar **relatórios simples** e personalizados.

Além disso, o projeto incorpora **gamificação**, **integração bancária**, **privacidade**, **assistente proativo**, **onboarding educativo**, **relatórios visuais** e o princípio de **design universal** para garantir uma boa experiência de usuário (UX) inclusiva e acessível.
